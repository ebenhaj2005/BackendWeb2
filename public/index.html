Dit zijn al mijn endpoints

  • GET /gebruiker - Haal alle gebruikers op
  • POST /gebruiker- Voeg een nieuwe gebruiker toe
  • GET /gebruiker/:id - Haal een specifieke gebruiker op
  • PUT /gebruiker/:id - Update een gebruiker
  • DELETE /gebruiker/:id - Verwijder een gebruiker
  • GET /gebruiker/10 - Haal gebruikers op met limit en offset
  • GET /nieuws - Haal alle nieuwsberichten op
  • POST /nieuws - Voeg een nieuwsbericht toe
  • GET /nieuws/:id - Haal een specifiek nieuwsbericht op
  • PUT /nieuws/:id - Update een nieuwsbericht
  • DELETE /nieuws/:id - Verwijder een nieuwsbericht
  • GET /nieuws/max - Haal gebruikers op met limit en offset
  • GET /gebruiker/search - Zoek gebruikers op voornaam
